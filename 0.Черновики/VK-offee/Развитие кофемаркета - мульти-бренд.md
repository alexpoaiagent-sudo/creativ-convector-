# Развитие кофемаркета: мульти-бренд и specialty кофе

## Содержание

**Идея:** Расширение ассортимента кофемаркета с введением нескольких обжарщиков и specialty кофе.

### Ключевые элементы:
1. **Мульти-бренд** — несколько обжарщиков на полках (не только один поставщик)
2. **Американо specialty дрип** — ассортимент из 6 кофе с оценкой 85+ баллов
3. **Станции для заваривания** — дрип-станции, изготовленные на лазерной резке
