# Тестовая заметка для проверки AI

Это тестовая заметка для проверки автоматического улучшения через OpenAI.

Заметка содержит несколько ошибок и неточностей которые AI должен исправить.

## Что должно произойти

1. Заметка отправится на GitHub
2. GitHub Actions запустит скрипт
3. OpenAI улучшит текст
4. Создастся Pull Request с улучшениями
