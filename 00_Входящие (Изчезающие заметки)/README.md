# Изчезающие заметки

## Назначение

Папка для сбора всех новых заметок в течение недели. Это точка входа для всех идей, мыслей и информации.

## Workflow

1. **В течение недели:** Создавайте заметки в этой папке
2. **В конце недели:** AI агент обрабатывает все заметки
3. **Результат:** Заметки превращаются в черновики с метаданными

## Что сюда писать

- Идеи для проектов
- Заметки с совещаний
- Мысли и инсайты
- Информация для обработки
- Любые записи, требующие категоризации

## Важно

- Эта папка **очищается еженедельно** после обработки
- Не храните здесь важные документы долго
- AI агент автоматически определит проект и роль для каждой заметки

## Команда для обработки

```bash
cd "/Users/alexander/Documents/Заметки"
npm run process
```

После обработки заметки переместятся в папку `01_Исчезающие черновики/` с добавленными метаданными.
