# Черновики

Сюда AI агент переносит обработанные заметки из "Исчезающих заметок".

## Структура

Каждый черновик привязан к приоритетному проекту:

```
2. Черновики/
├── VK-Coffee/
├── Marathon-v2/
├── Creative-Convector/
└── [Другие проекты]/
```

## Что содержит черновик

- **Шапка (frontmatter):**
  - `role:` - Роль по FPF (F1-F9)
  - `project:` - Название проекта
  - `status:` - Статус (черновик)
  - `created:` - Дата создания
  - `updated:` - Дата обновления

- **Содержание:**
  - Исходный текст заметки
  - Ключевые идеи
  - Рекомендации AI агента

## Работа с черновиками

1. Просмотрите рекомендации AI
2. Доработайте содержание
3. Переместите в соответствующий проект

---

**Предыдущий шаг:** ← `1. Исчезающие заметки`
**Следующий шаг:** → `3. Приоритетные проекты`
