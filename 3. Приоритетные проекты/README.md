# Приоритетные проекты

Здесь хранятся все активные проекты, организованные по методологии FPF.

## Структура проекта

Каждый проект организован по таблице 3×3 (9 ролей):

```
Проект/
├── F1-Контекст/          (Предприниматель + Надсистема)
├── F2-Окружение/         (Инженер + Надсистема)
├── F3-Взаимодействие/    (Менеджер + Надсистема)
├── F4-Требования/        (Предприниматель + Целевая система)
├── F5-Архитектура/       (Инженер + Целевая система)
├── F6-Реализация/        (Менеджер + Целевая система)
├── F7-Принципы/          (Предприниматель + Система создания)
├── F8-Платформа/         (Инженер + Система создания)
└── F9-Команда/           (Менеджер + Система создания)
```

## Таблица 3×3 (FPF)

```
                     Предприниматель  Инженер           Менеджер
                     (Зачем?)         (Как устроено?)   (Как работает?)
┌────────────────────────────────────────────────────────────────────┐
│ Suprasystem        F1               F2                F3           │
│ (Надсистема)       Контекст         Окружение         Взаимодействие│
├────────────────────────────────────────────────────────────────────┤
│ System-of-Interest F4               F5                F6           │
│ (Целевая система)  Требования       Архитектура       Реализация   │
├────────────────────────────────────────────────────────────────────┤
│ Constructor        F7               F8                F9           │
│ (Система создания) Принципы         Платформа         Команда      │
└────────────────────────────────────────────────────────────────────┘
```
