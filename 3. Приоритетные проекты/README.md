# 3. Приоритетные проекты

Финальные, структурированные материалы по каждому проекту.

## Структура (FPF - First Principles Framework)

Каждый проект организован по 9 ролям (таблица 3x3):

|              | Предприниматель (Зачем?) | Инженер (Как устроено?) | Менеджер (Как работает?) |
|--------------|--------------------------|-------------------------|--------------------------|
| Надсистема   | F1 - Контекст            | F2 - Окружение          | F3 - Взаимодействие      |
| Целевая сист.| F4 - Требования          | F5 - Архитектура        | F6 - Реализация           |
| Конструктор  | F7 - Принципы            | F8 - Платформа          | F9 - Команда              |

## Проекты

- **VK-Coffee** - Кофейный маркет / сеть кофеен
- **Marathon-v2** - Программа адаптации сотрудников
- **Creative-Convector** - Система управления заметками (этот проект)

## Как добавить проект

Новые проекты создаются автоматически при распределении заметок (`AI: Distribute Notes`).
Или создайте вручную папку с 9 подпапками F1-F9.
