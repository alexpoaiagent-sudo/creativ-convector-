# Архив

## Назначение

Хранилище для завершенных, неактуальных или устаревших черновиков и заготовок.

## Что сюда попадает

- Завершенные черновики, которые уже использованы
- Неактуальные идеи
- Устаревшая информация
- Материалы, которые больше не нужны, но жалко удалять

## Структура

Рекомендуется организовывать архив по датам или проектам:

```
99_Архив/
├── 2026-01/
├── 2026-02/
└── По проектам/
    ├── ВК кофе/
    └── Бариста класс/
```

## Важно

- Архив не обрабатывается AI агентом
- Файлы здесь не участвуют в анализе покрытия таблицы 3×3
- Периодически очищайте архив от действительно ненужных материалов

## Восстановление

Если материал из архива снова стал актуален:
1. Переместите его обратно в `01_Исчезающие черновики/`
2. Обновите метаданные (дату `updated`)
3. Доработайте и перенесите в проект
