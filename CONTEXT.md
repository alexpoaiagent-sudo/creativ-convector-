## AI АГЕНТЫ

### Команды в Obsidian (Cmd+P)
| Команда | Что делает |
|---------|-----------|
| `AI (Claude): Enhance Note` | Анализ заметки через Claude |
| `AI (ChatGPT): Enhance Note` | Анализ заметки через ChatGPT |
| `AI (Claude): Create Analysis` | Анализ отчёта через Claude |
| `AI (ChatGPT): Create Analysis` | Анализ отчёта через ChatGPT |
| `AI: Distribute Notes` | Автораспределение заметок |
| `Generate Weekly Report` | Недельный отчёт |
| `Git Commit & Push` | Коммит и пуш |
| `Git Status` | Статус git |

### Распределение заметок
```bash
python3 .github/scripts/distribute_notes.py
```
- Читает заметки из "1. Исчезающие заметки/"
- Определяет проект по ключевым словам
- Создает frontmatter с метаданными
- Перемещает в "2. Черновики/{Проект}/"
