# Creative Convector — Obsidian Vault

Хранилище заметок с синхронизацией в GitHub для работы с AI-агентами.

## Структура

### 0.Входящие
Новые заметки, которые нужно обработать AI-агентом.

**Workflow:**
1. Создаёте заметку в этой папке
2. AI-агент читает заметку
3. AI-агент раскидывает по проектам или черновикам

### 0.Черновики
Черновики, над которыми работает AI-агент.

**Подпапки:**
- `VK-offee/` — черновики для сети кофеен «Вкусный Кофе»
- `Бариста-класс/` — черновики для проекта обучения бариста
- `ИТ-проекты/` — черновики для IT-проектов
- `general/` — общие черновики

**Примечание:** AI-агент автоматически создаёт новые папки для новых проектов при обработке заметок.

### 0.Архив
Завершённые или неактуальные заметки.

### Изчезающие заметки
Временные заметки (существующая папка).

### Черновики по приоритетным проектам
Приоритетные черновики (существующая папка).

## Синхронизация с GitHub

### Автоматическая (через Obsidian Git plugin)
- Автокоммит каждые 10 минут
- Автопуш каждые 30 минут
- Автопулл при открытии Obsidian

### Ручная
```bash
cd ~/Documents/Obsidian\ Vault
git add .
git commit -m "Update notes"
git push origin main
```

## Работа с AI-агентом

### Команды для AI

**Обработать входящие:**
```
Обработай заметки из папки 0.Входящие и раскидай их по проектам
```

**Создать черновик:**
```
Создай черновик для [проект] на основе заметки [название]
```

**Синхронизировать с проектом:**
```
Синхронизируй черновики из 0.Черновики/VK-offee/ с репозиторием VK-offee
```
