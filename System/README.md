# System (Системные папки)

Служебные файлы и конфигурация.

## Содержимое

- `Архив/` - Архивированные заметки (старше 7 дней)
- `Исходники/` - Исходные тексты заметок по неделям
- `Вложения/` - Изображения, PDF и другие файлы
- `Отчёты AI/` - Отчёты о работе AI агента
- `Конфигурация/` - Настройки системы

## Примечание

Эта папка скрыта из основной навигации Obsidian.
Для доступа используйте поиск или прямые ссылки.
