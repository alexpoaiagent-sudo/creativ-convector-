# Приоритетные проекты

## Назначение

Главная папка для всех приоритетных проектов. Каждый проект организован по методу SRT (таблица 3×3) с подпапками F1-F9.

## Структура

Каждый проект содержит 9 папок, соответствующих ячейкам таблицы 3×3:

### Надсистема (Suprasystem)
- **F1-Контекст-и-рынок** — Зачем? (Предприниматель)
- **F2-Окружение-и-интерфейсы** — Как устроено? (Инженер)
- **F3-Партнёрства** — Как работает? (Менеджер)

### Целевая система (System-of-Interest)
- **F4-Ценности-и-описание** — Зачем? (Предприниматель)
- **F5-Меню-и-продукты** — Как устроено? (Инженер)
- **F6-Операции-и-качество** — Как работает? (Менеджер)

### Система создания (Constructor)
- **F7-Финансы-и-принципы** — Зачем? (Предприниматель)
- **F8-Поставщики-и-инструменты** — Как устроено? (Инженер)
- **F9-Команда-и-обучение** — Как работает? (Менеджер)

## Проекты

1. **ВК кофе (сеть кофеен)** — Развитие сети кофеен
2. **Бариста класс** — Обучающие программы для бариста
3. **Парк** — Проект парка
4. **Стекляшка** — Проект стекляшки
5. **Стратегирование** — Долгосрочные приоритеты
6. **Саморазвитие** — Личное развитие

## Workflow

1. Черновики из `01_Исчезающие черновики/` дорабатываются
2. Готовые черновики становятся "заготовками"
3. Заготовки размещаются в соответствующей папке F1-F9 проекта

```bash

```
