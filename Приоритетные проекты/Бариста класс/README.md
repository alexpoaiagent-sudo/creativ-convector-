# Бариста класс

## Описание проекта

Образовательный проект по обучению бариста — создание программ, курсов и материалов для профессионального развития специалистов кофейной индустрии.

## Цели

- Создание качественных обучающих программ
- Подготовка профессиональных бариста
- Развитие кофейной культуры
- Сертификация и аттестация

## Структура по таблице 3×3

### Надсистема (Suprasystem)
- **F1-Контекст-и-рынок** — Рынок образования, конкуренты, спрос
- **F2-Окружение-и-интерфейсы** — Партнеры, площадки, оборудование
- **F3-Партнёрства** — Сотрудничество с кофейнями, поставщиками

### Целевая система (System-of-Interest)
- **F4-Ценности-и-описание** — Ценности обучения, методология
- **F5-Меню-и-продукты** — Программы курсов, модули, материалы
- **F6-Операции-и-качество** — Процесс обучения, практика, оценка

### Система создания (Constructor)
- **F7-Финансы-и-принципы** — Ценообразование, экономика курсов
- **F8-Поставщики-и-инструменты** — Оборудование для обучения, платформы
- **F9-Команда-и-обучение** — Преподаватели, наставники, методисты

## Связь с другими проектами

- **ВК кофе** — выпускники курсов работают в сети
- **Саморазвитие** — личное развитие как преподавателя
