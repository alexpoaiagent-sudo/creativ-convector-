# Стратегирование

## Описание проекта

Долгосрочное стратегическое планирование и развитие. Работа над приоритетами, целями и направлениями развития на длительную перспективу.

## Цели

- Определение долгосрочных приоритетов
- Стратегическое планирование
- Анализ возможностей и рисков
- Принятие ключевых решений

## Структура по таблице 3×3

### Надсистема (Suprasystem)
- **F1-Контекст-и-рынок** — Макротренды, возможности, угрозы
- **F2-Окружение-и-интерфейсы** — Экосистема, связи, влияние
- **F3-Партнёрства** — Стратегические альянсы, сотрудничество

### Целевая система (System-of-Interest)
- **F4-Ценности-и-описание** — Видение, миссия, ценности
- **F5-Меню-и-продукты** — Портфель проектов, инициативы
- **F6-Операции-и-качество** — Процессы принятия решений, контроль

### Система создания (Constructor)
- **F7-Финансы-и-принципы** — Инвестиционная стратегия, принципы
- **F8-Поставщики-и-инструменты** — Инструменты стратегирования, методологии
- **F9-Команда-и-обучение** — Развитие компетенций, обучение

## Связь с другими проектами

Стратегирование влияет на все остальные проекты:
- Определяет приоритеты развития
- Задает направление движения
- Координирует ресурсы
